# ICAD — Institutional Constitutional Agentic Design

A platform for running ACCT (Agentic Constitutional Commitment Typology) protocols with AI agents operating within constitutionally enforced boundaries.

## First Deployment

**Process:** Carta de Disponibilidad de Agua Potable (Water Availability Letter)  
**Institution:** ASADA Playas de Nosara, Costa Rica  
**Status:** Pre-build — documentation and architecture complete
